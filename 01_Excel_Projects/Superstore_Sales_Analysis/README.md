## Project Overview

This project analyzes Superstore sales data using Microsoft Excel. The objective of this project is to clean raw sales data, perform analysis, create visualizations, and generate business insights using Excel features.

## Tools Used

- Microsoft Excel for Web
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Data Visualization

## Dataset Description

The dataset contains sales transactions with information about:

- Order ID
- Order Date
- Ship Date
- Customer Details
- Segment
- Country
- State
- Market
- Region
- Product Details
- Category
- Sales
- Quantity
- Discount
- Profit
- Shipping Cost
- Order Priority
- Year

## Data Cleaning Process

The following cleaning steps were performed:

- Organized raw data into a structured table
- Checked and corrected data formats
- Converted date columns into proper date format
- Applied correct formats for sales, profit, and discount values
- Verified data consistency

## Analysis Performed

### Key Performance Indicators (KPIs)

Calculated:

- Total Orders
- Total Sales
- Total Profit
- Total Quantity Sold
- Average Discount

### Pivot Table Analysis

Created PivotTables to analyze:

1. Sales by Category
2. Profit by Region

### Dashboard

Created an interactive dashboard containing:

- KPI summary
- Sales analysis chart
- Profit analysis chart
- Business insights

## Key Insights

- Technology category generated the highest sales with $4.74 million, showing strong customer demand.
- Furniture category was the second-highest revenue contributor with $4.11 million.
- Central region recorded the highest profit contribution.
- Canada recorded the lowest profit contribution among regions.
- The business achieved total sales of $12.64 million with $1.46 million profit.
- Average discount was 14.29%, which should be monitored to maintain profitability.
